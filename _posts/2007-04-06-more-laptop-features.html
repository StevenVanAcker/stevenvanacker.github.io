--- layout: post title: more laptop features date: '2007-04-06T19:23:00.000+02:00' author: Steven Van Acker tags: modified_time: '2008-01-18T10:44:23.837+01:00' blogger_id: tag:blogger.com,1999:blog-5513234137363262204.post-4535220370988409909 blogger_orig_url: http://www.singularity.be/2007/04/more-laptop-features.html --- Here are some more features I want on my laptop:


  • Dual screen for programming and presentations and the like.

  • Disable the annoying touchpad sensitivity. I don't want to click on anything or select anything when I tap my touchpad too hard.

  • 3D support. I have an ATI X300 graphics card and plan to use it.



I'm also thinking about making a virtual Ubuntu package that I can install on a fresh ubuntu installation, and that will install and configure all my laptop features automatically. in case I ever get a new laptop, I don't have to do all this work again.

Also, my timesheet system (the one that keeps track of what projects I work on) should keep track of when I lock my desktop (when I'm on break) and it should work together with my local calendar system somehow.
Being able to synch with another calendar system would be a big plus (like MS Exchange for example, or whatever other expensive bloated closed-source system my employer plans to buy)
