--- layout: post title: OpenVPN looks nice date: '2007-04-21T00:02:00.000+02:00' author: Steven Van Acker tags: modified_time: '2008-01-18T10:47:28.236+01:00' blogger_id: tag:blogger.com,1999:blog-5513234137363262204.post-3061565218900790032 blogger_orig_url: http://www.singularity.be/2007/04/openvpn-looks-nice.html --- OpenVPN is SSL-VPN software that can do layer2/layer3 tunneling over a single udp/tcp port. I'm impressed.

In addition, Ubuntu has a plugin for it's vpnc client, so that an OpenVPN client would integrate nicely.

The only thing I don't like, it the lengthy configuration cycle: creating a CA, creating keys, configuring this configuring that. And I didn't get to the actual configuration file yet!

There is an #openvpn channel on freenode where they tolerated my rant. So I'm impressed with their attitude aswell. I think it's well worth the effort to dig into OpenVPN.
